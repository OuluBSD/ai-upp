Class WeakPtr<T>

WeakPtr - Weak reference that doesn't affect reference counting

Holds a weak reference to a value. The value may be deleted even if WeakPtr exists. Must be upgraded to Ptr to access.

Example

const ptr = new Ptr({ x: 10 });
const weak = ptr.ToWeak();

const locked = weak.Lock();
if (locked.HasValue()) {
  console.log(locked.Get().x); // 10
}
