# PowerShell Transpiler Support

Implement non-typed PowerShell (`.ps1`) output.
Supporting script automation for Windows environments.

## Target: Windows GUI
- Transpile `.py` + `.form` to a **PowerShell script** (`.ps1`).
- Implement GUI using **XAML/WPF** (Windows Presentation Foundation) for rich UI interactions.
- Map `.form` layout components to XAML code.
