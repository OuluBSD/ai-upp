# Task: Implement Window and Layout Logic

## Goal
Implement the **Window** menu logic to manage panes, toolbars, and layout configurations.

## Strategy
1.  **Pane Toggles**: Connect menu items to `Show()`/`Hide()` methods of each dockable pane.
2.  **Layout Presets**: Implement "Default", "Rstudio", and "Matlab" layout presets by manipulating the docking state.
3.  **Fullscreen**: Implement F11 toggle for fullscreen mode.

## Success Criteria
- [ ] Panes can be toggled on/off from the menu.
- [ ] Switching to "Default layout" resets the docking configuration.
- [ ] "Fullscreen mode" correctly expands the window.
