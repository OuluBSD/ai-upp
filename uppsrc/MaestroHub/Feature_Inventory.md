# Maestro Feature Inventory

## CLI Commands (MaestroCLI)
- `init`: Bootstrap repo.
- `repo`: Scan/List packages.
- `plan`: Manage tracks/phases/tasks.
- `tu`: Translation Unit intelligence.
- `log`: Scan logs for findings.
- `issues`: Manage Maestro issues.
- `work`: Start/Manage work sessions.
- `runbook`: Create/Execute runbooks.
- `workflow`: Edit/Visualize PUML.
- `settings`: Configure Maestro.
- `ops`: System health and automation.

## Automation & Orchestration
- `ConversionOrchestrator`: Batch conversion logic.
- `PlaybookManager`: Management of reusable instructions.
- `EvidencePack`: Automated state collection for audit.
- `RegressionReplay`: Systematic verification of AI fixes.

## Multi-Project/Workspace
- Project Indexing: Global TU and Issue database.
- Batch Log Scanning: Scanning across directory trees.
- Runbook Library: Centralized one-time designs for repeated use.
