# Software Protection and the Tuples of Core
**Date Span:** 2010-09-01 to 2010-09-30

### Core Expressiveness
Introduced the `Tuple` template to the core library and added `ReverseFind` to strings. A significant ergonomic update ensured all container `Add` methods return a reference (`T&`) to the newly created element.

### Security and Encryption
Launched the **Protect** package in the Bazaar for software obfuscation and Snow2-based code encryption. This was complemented by the `StreamCypher` package for robust stream data security.

### UI and Help Systems
Introduced `HelpViewer` with native Table of Contents support. The `Animate` system was refactored for smoother dropdown and popup transitions, and `GridCtrl` received property-grid and sync improvements.

### Professional Imaging and Networking
Hardened DPI metadata handling for PNG, TIFF, and JPEG plugins and matured the `RasterPlayer` demo. `XmlRpc` gained improved `int64` support, and PostgreSQL auto-reconnect logic was significantly hardened.
