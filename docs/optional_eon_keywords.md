# Optional eon keywords

### Copy-paste list
AUDIO BUILTIN_PORTAUDIO CAMERA DX10 DX11 DX12 DX9 FBO FLUIDLITE FLUIDSYNTH GUBO HACK HAL LOCALHMD LV2 MEDIA MIDI OGL OPENCV OPENHMD PORTAUDIO SDL2 SOFTHMD UWP VIDEO VOLUMETRIC VR X11 

### Explanations

#### AUDIO
Generic audio code.

##### BUILTIN_PORTAUDIO
Compile builtin portaudio in uppsrc/plugin/portaudio instead of system portaudio.

##### CAMERA 
Generic camera code.

##### DX10
DirectX10 support.

##### DX11 
DirectX11 support.

##### DX12 
DirectX12 support.

##### DX9
DirectX9 support.

##### FBO
Framebuffer Object. This is needed for 3D graphics.

##### FLUIDLITE
Compile and use builtin fluidsynth without features which has dependencies.

##### FLUIDSYNTH 
Use fluidsynth.

##### GUBO
Use our graphical cuboid system (3d with u++ CtrlLib like syntax).

##### HACK
Generic flag for very experimental code.

##### HAL
Hardware Abstraction Layer. Needed when using all-in-one libraries like SDL or Allegro or Microsoft equivalents.

##### LOCALHMD
Compile builtin openhmd with a lot of changes (fork basically).

##### LV2
Support for LV2 effects and instruments.

##### MEDIA
Generic media support for media files and streams.

##### MIDI
Generic midi support.

##### OGL
Generic OpenGL support.

##### OPENCV
Use OpenCV libraries.

##### OPENHMD
Use OpenHMD.

##### PORTAUDIO
Use PortAudio audio inputs and outputs

##### SDL2 
Use SDL2 all in one libray for low-level games and applications.

##### SOFTHMD
Use our local effort to support VR HMD

##### UWP
Universal Windows Program. Needed for Windows Store programs.

##### VIDEO 
Generic Video support.

##### VOLUMETRIC 
Generic Volumetric data support.

##### VR 
Generic Virtual Reality support.

##### X11 
Support X11 graphics (posix).
