Class Gate

A Gate class similar to U++ Gate, providing a simple boolean-based gate mechanism. The gate can be opened or closed and can be tested to see if it's open.

Example

const gate = new Gate();
console.log(gate.IsOpen()); // false

gate.Open();
console.log(gate.IsOpen()); // true

gate.Close();
console.log(gate.IsOpen()); // false
