# VideoCardCandidateExtractor

Consumes 0253 event output and 0252 enrichment, producing conservative video-card candidates and a review gallery. Keep classification evidence-based: semantic hits and geometry may promote a candidate, but missing evidence must remain `unknown`.

Build with `bin\\build.exe -m MSVS22x64 .\\reference\\VideoCardCandidateExtractor\\VideoCardCandidateExtractor.upp`.
