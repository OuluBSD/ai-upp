# The Gtk Backend and the Eigen Promotion
**Date Span:** 2012-12-01 to 2012-12-31

### Rainbow Milestone: Gtk Backend
Launched the intensive development of the **Gtk backend** for the Rainbow project. This new architecture introduced native image handling, skinning support, and foundational implementation for primary selections and **Drag & Drop**, signaling a major step toward decoupled GUI abstraction.

### Eigen Matrix Library Promotion
Formally promoted the **Eigen** library from a plugin to the canonical `uppsrc` collection. Upgraded to version 3.1.2, this move solidified U++ as a top-tier platform for high-performance linear algebra and advanced mathematical algorithms.

### Professional UI and Scientific Polish
**ScatterDraw** and **ScatterCtrl** received significant updates and the start of a comprehensive documentation drive. **Controls4U** achieved a translation milestone, ensuring consistent multilingual support across all specialized UI controls.

### Database and Tooling Refinement
Added support for **PostgreSQL $$ notation** in SQL scripts and introduced the **DRAWTEXTLINES** option for fine-grained text rendering control. TheIDE was refined with automatic configuration backups and the addition of the **Dutch (nl-nl)** translation.
