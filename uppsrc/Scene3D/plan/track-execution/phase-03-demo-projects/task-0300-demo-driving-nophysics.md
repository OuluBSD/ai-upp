# Task 0300 - Demo: Driving (No Physics)

## Goal
Create a simple driving demo with discrete move/turn modes.

## Scope
- One box body + four cylinder wheels.
- Plane ground, two checkpoints, one goal.
- Straight/turning modes only (no physics).
- 3 laps required to finish, then exit.

## Success Criteria
- Demo runs in executor and ModelerApp execution.
- Lap counter increments with checkpoints.
- Exits after 3 laps.
