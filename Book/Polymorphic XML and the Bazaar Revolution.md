# Polymorphic XML and the Bazaar Revolution
**Date Span:** 2010-01-01 to 2010-01-31

### PolyXML Serialization
Introduced `PolyXML`, a polymorphic XMLization system enabling the serialization of complex C++ object hierarchies using a central `ClassFactory`. This significantly simplified the handling of runtime-dynamic types.

### Linux Notifications and Bazaar Growth
Achieved passive notification parity on Linux by integrating `libnotify` into the core `TrayIcon` class. The Bazaar expanded with `TabBarCtrl` and saw a major documentation drive for `UnitTest++`.

### Networking and Security
Landed support for **Digest Authentication**, providing enhanced security for HTTP clients. Ftp progress callbacks were standardized to the `Gate2` paradigm, and `FindFile` was fixed for i18n characters in POSIX.

### Help System and UltimateBook
TheIDE's help system gained match-highlighting and result navigation. The "UltimateBook" project was launched to provide comprehensive written documentation, and `Painter` completed its `DrawImageOp` implementation.
