# VB6 Transpiler Support

Implement non-typed VB6 output.
Supporting BASIC syntax, including `Sub`, `Function`, and VB6 primitives.

## Full Project Integration
- Transpile `.py` + `.form` to a complete, runnable **VB6 Project** (`.vbp`).
- Map `.form` files back to VB6 `.frm` files.
- Support all VB6 file formats: `.cls`, `.bas`, `.csi`, `.exp`, `.lvw`, `.vbp`, `.vbw`, `.frm`, `.frx`, `.res`.
