Class Mutex

Mutex (Mutual Exclusion) class for synchronization between threads This implementation uses async-mutex library to provide mutex functionality in a U++-like API

Example

const mutex = new Mutex();

async function criticalSection() {
  const release = await mutex.Lock();
  try {
    // Critical section code here
    console.log('In critical section');
  } finally {
    release(); // Unlock
  }
}
