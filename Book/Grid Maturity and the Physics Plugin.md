# Grid Maturity and the Physics Plugin
**Date Span:** 2010-07-01 to 2010-07-31

### GridCtrl and Real-time Physics
`GridCtrl` reached a major milestone in programmability with the addition of `SetValues`, `GetValues`, and `SetCtrl`, while the **Box2D** physics library was added as a core plugin to enable rigid-body simulations within U++ applications.

### Database Null-Safety and Parity
Introduced the `IsSame` operator for null-safe equality testing in PostgreSQL and MySQL. Landed support for Oracle-style multi-column `SET` in `UPDATE` statements and standardized PostgreSQL's `Like` operator to be case-sensitive.

### OLE Interop and Advanced UI
Enabled basic conversion between U++ `ValueArray` and Windows OLE `SAFEARRAY`. `TabBar` and `Docking` received visual and functional overhauls, including alternative close icons and a new "SortTitles" organization logic.

### Framework Hardening and Tooling
TheIDE added support for Windows SDK 7.1. Core library enhancements included `Time::Compare` and better whitespace handling in `XmlParser`. The `ndisasm` plugin was upgraded, and `XmlRpc` gained proxy authentication support.
