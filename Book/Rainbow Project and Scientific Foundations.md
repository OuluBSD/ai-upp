# Rainbow Project and Scientific Foundations
**Date Span:** 2011-06-01 to 2011-06-30

### Rainbow Project Launch
Commenced development of the **Rainbow** architecture, a unified GUI abstraction layer. The initial focus was on the **Framebuffer (FB) backend**, enabling U++ applications to run on headless systems or embedded devices without X11 or Win32.

### Scientific and Math Powerhouses
The Bazaar welcomed **Eigen** (matrix algebra) and **OpenCV** (computer vision) packages. These integrations transformed U++ into a viable platform for scientific computing, robotics, and advanced image processing.

### Professional Tooling: Open Watcom
Added support for the **Open Watcom** builder in TheIDE, expanding the range of supported compiler toolchains. The topic editor and include-assist were also refined for better developer productivity.

### Core Hardening and Windows 7
`CtrlCore` now enables flicker-free backpainting by default on Windows 7 and newer. The Oracle backend was significantly upgraded with `OracleClob` for character large-object handling, and `HttpClient` added support for the HTTP 100 'Continue' status.
