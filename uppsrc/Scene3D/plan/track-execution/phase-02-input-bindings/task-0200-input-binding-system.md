# Task 0200 - Input Binding System

## Goal
Expose keyboard and mouse input to execution scripts.

## Scope
- Bind key down/up and mouse events to script callbacks.
- Provide input state queries (pressed, released, axis delta).
- Ensure bindings work in ModelerApp execution and ModelMediaShell.

## Success Criteria
- Scripts can read key and mouse state each frame.
- Event callbacks fire for key and mouse input.
- Demo projects can control movement and actions via input.
