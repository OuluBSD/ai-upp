# Task 0600 - Skeleton Rigging

## Goal
Add skeleton creation and rigging tools for meshes.

## Scope
- Create/edit skeleton hierarchy (bones, joints) in the scene tree.
- Visualize skeleton in viewport (axes, bone lines).
- Attach skeletons to mesh objects.

## Implementation Notes
- Store skeleton as a component (VfsValueExt) on mesh objects or as a separate node.
- Provide transform gizmos for bones.

## Success Criteria
- Skeleton hierarchy can be created and edited in the UI.
- Bones are visible and selectable in the viewport.
