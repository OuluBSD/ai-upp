# Task 1302 - Timeline Copy/Paste

## Goal
Enable copy/paste of keyframes and ranges.

## Scope
- Copy keyframes on current row or selected rows.
- Paste at playhead or target frame.
- Paste preserves relative offsets inside selection.
- Support multiple channels (position/orientation/mesh/2D).

## Success Criteria
- Copy/paste works for single and multi-row selections.
- Paste does not corrupt unrelated rows.
