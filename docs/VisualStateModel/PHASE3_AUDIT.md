## Blocking Issues for Phase 4

None blocking. The adapter sequence recommended in `VIDEO_SOURCE_ADAPTERS.md`
is valid:

1. `VsmSessionStoreSource` ‚Äî replay existing sessions (task 0022)
2. Image sequence importer (task 0023)
3. Capture sink and recording session writer (task 0024)
4. Workbench source open/import flow (task 0025)
5. Annotation overlay authoring tools (task 0026)
6. MJPEG-over-HTTP prototype (task 0027)

JPEG/PNG decode is not available headlessly yet. Phase 4 imports `.vsm` files
only; JPEG/PNG import is deferred to a follow-up dependency.
