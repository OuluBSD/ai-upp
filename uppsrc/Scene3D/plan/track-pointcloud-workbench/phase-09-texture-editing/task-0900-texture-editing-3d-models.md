# Task 0900 - Texture Editing for 3D Models

## Goal
Add texture painting/editing tools for 3D models.

## Scope
- Texture paint brush with size/strength/color.
- Basic projection modes (UV paint, screen paint).
- Save edited textures to project data_dir.

## Implementation Notes
- Start with UV paint on existing UVs.
- Use a texture layer stack for edits.

## Success Criteria
- User can paint on a model and see texture updates in viewport.
- Edited textures are saved and reloaded.
