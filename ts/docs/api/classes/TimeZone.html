Class TimeZone

TimeZone class for time zone handling with U++-like API Provides time zone operations similar to U++ TimeZone functionality

Example

const timeZone = new TimeZone('UTC');
console.log(timeZone.GetName()); // UTC

const localTimeZone = TimeZone.GetLocal();
console.log(localTimeZone.GetName());
