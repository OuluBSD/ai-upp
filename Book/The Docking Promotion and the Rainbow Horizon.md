# The Docking Promotion and the Rainbow Horizon
**Date Span:** 2011-09-01 to 2011-09-30

### Docking System Maturity
The sophisticated **Docking** window management system was officially promoted from the Bazaar to the canonical `uppsrc` package collection. This transition provided all U++ applications with a standardized, professional-grade solution for complex multi-pane layouts.

### Rainbow Backend Expansion
Initial functional tests for the **LinuxFb** (Framebuffer) and **SDLFb** backends marked a significant diversification of the Rainbow project. This expansion reinforced the framework's commitment to becoming a universal GUI toolkit for both accelerated desktops and embedded devices.

### Objective C and IML Design
TheIDE added native support for **Objective C**, broadening its appeal for cross-platform developers. The image list (.iml) designer was upgraded with powerful import/export capabilities, and the BRC tool was refactored to bypass legacy compiler size limits.

### Core and Security Robustness
`String` gained `GetCharCount`, and the `Format` engine was fixed for precise UTF-8 padding. The **Protect** package became fully multi-thread safe, **PolyXML** added support for polymorphic maps, and `HttpClient` implemented the `PUT` method.
