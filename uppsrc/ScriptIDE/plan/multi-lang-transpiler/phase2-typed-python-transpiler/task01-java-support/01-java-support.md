# Java Transpiler Support

Implement Java output for the typed-python transpiler.
Mapping Python's `typing` primitives to Java types.

## Target: Desktop (SWT)
- Generate Java code that uses the **SWT (Standard Widget Toolkit)**.
- Map `.form` layout components to SWT widgets.

## Target: Android
- Generate an **Android Studio** compatible Java project.
- Map `.form` layout components to Android XML layouts or programmatic UI.
