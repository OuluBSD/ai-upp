# PHP Transpiler Support

Implement non-typed PHP output.
Supporting PHP 7/8 syntax for web-side scripts.

## Target: Web Page
- Transpile `.py` + `.form` to a **PHP Web Page**.
- Static layout based on `.form` structure.
- Implementation of dynamic features using **JavaScript**.
- Mapping Python logic to PHP (server-side) and JS (client-side) where appropriate.
