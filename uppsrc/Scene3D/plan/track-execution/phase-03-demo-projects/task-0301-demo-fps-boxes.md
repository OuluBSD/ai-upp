# Task 0301 - Demo: FPS (Boxes)

## Goal
Create a simple FPS-style demo with box geometry and projectiles.

## Scope
- Plane ground and box houses.
- Box gun, box projectiles.
- Target is a red box; game ends on hit.

## Success Criteria
- Player moves between houses and shoots.
- Red target box hit ends the game.
- Runs in executor and ModelerApp execution.
