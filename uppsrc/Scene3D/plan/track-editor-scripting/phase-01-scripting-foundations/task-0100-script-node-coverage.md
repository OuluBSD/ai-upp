# Task 0100 - Script Node Coverage

## Goal
Make scripting available as reusable components across the scene graph, not just for a single object.

## Scope
- Allow script components on directories and/or scene root.
- Add creation of script components from tree/context menus.
- Ensure script editor is accessible from Properties.

## Success Criteria
- Scripts can attach to both objects and directories.
- Context menus allow creating script components.
- Scripts are persisted and reloaded with the scene.
