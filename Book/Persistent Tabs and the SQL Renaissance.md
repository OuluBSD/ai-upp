# Persistent Tabs and the SQL Renaissance
**Date Span:** 2009-06-01 to 2009-06-30

### SQL Education and Connectivity
Completed a comprehensive SQL tutorial and added PostgreSQL support to `SqlCommander`. Refined `SqlBinary` to correctly handle empty values as NULL and cleaned up the core database package structure.

### TheIDE Persistence and Automation
Introduced persistent file tabs across sessions and support for multiline commit messages in SVN. Automated package population by implementing a directory source tree import feature and extended Assist++ to handle `.sch` schema files.

### RichText and UI Promotions
Promoted `MultiList` to `uppsrc` and overhauled `RichText` painting to use `RichObject::ToImage`, resolving transparency issues with embedded objects. Introduced a new 'text' rich object format and improved `TabBar` integration with `Docking`.

### Platform and Core Stability
Addressed critical SCIM/XIM focus issues and Compiz visual artifacts in Linux/X11. Fixed `DirectoryExists` for Win32 drives and standardized CRLF line endings in XML output. Expanded the Bazaar with `BlueBar` and `Skulpture` theme updates.
