# GTK Native Choosers and Cairo Graphics
**Date Span:** 2013-03-01 to 2013-03-31

### GTK Desktop Parity
`FileSel` achieved seamless Linux integration with support for the native **GTK+ file chooser**. This was accompanied by the implementation of **GLCtrl for GTK**, enabling U++ applications to host hardware-accelerated OpenGL content within native Gtk windows.

### Cairo-Powered Vector Graphics
The **Rainbow** software backends were bolstered with high-performance **Cairo drawing operations**, including support for line dashing, arcs, and complex poly-polygons. This graphical push was supported by a major documentation drive for the **SortedMap** family.

### Core Utilities and Safety
Introduced the `decode` multi-mapping utility and expanded **min/max** to support 3 and 4 parameters. The `Color` class was hardened against accidental boolean conversions, and the **INI parser** was refactored for better resilience.

### IDE and Professional Polish
The **ESC** macro engine received logic enhancements, and `RichEdit` switched to the standard **I-Beam cursor**. `EditField` gained **ShowSpaces** and custom highlighting callbacks, while the **Report** package added support for printing hints.
