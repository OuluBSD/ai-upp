# Animated GIFs and the QTF Designer
**Date Span:** 2010-06-01 to 2010-06-30

### Animated Graphics and Multi-page Rasters
Introduced native support for **Animated GIFs** and a unified multi-page raster architecture. This enabled sophisticated image sequence handling with support for sub-rectangles and frame disposal methods, showcased in the `RasterMultiPage` example.

### Tooling: QTF Designer and Smarter Selector
Launched the visual **QTF Designer** in TheIDE, simplifying the creation of rich text resources. The package selector was optimized to sort by "closeness" to the main package directory, prioritizing relevant subpackages and nests.

### System Monitoring and Includes
`SysInfo` was expanded with CPU temperature and battery monitoring via ACPI. Standardized MAC address retrieval was added, and a concerted effort was made to resolve header interferences between U++ and system Xlib libraries.

### Core and Educational Growth
`bool` became a first-class `Value` type, and strings gained `ReverseFind`. The official website peak with the launch of a comprehensive Tutoring manual and video tutorials, alongside the addition of Simplified Chinese translations.
