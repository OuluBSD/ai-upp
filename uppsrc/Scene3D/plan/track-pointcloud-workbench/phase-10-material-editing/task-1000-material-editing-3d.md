# Task 1000 - Material Editing for 3D Models

## Goal
Provide a material editor for 3D model materials.

## Scope
- Edit base color, metallic, roughness, emissive, normal maps.
- Material previews and assignment to model submeshes.
- Save material settings to project.

## Success Criteria
- Material changes reflect in viewport immediately.
- Materials persist through save/load.
