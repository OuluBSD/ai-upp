# Task 0900 - Scene-Level Timelines

## Goal
Add per-scene timelines that can drive objects and components independently from the global timeline.

## Scope
- Introduce a Scene Timeline component (e.g., GeomSceneTimeline).
- Allow scene-level playback/looping separate from global play.
- Expose scene timeline in UI.

## Success Criteria
- Scene timeline can be edited and played independently.
- Scene timeline state persists in save/load.
