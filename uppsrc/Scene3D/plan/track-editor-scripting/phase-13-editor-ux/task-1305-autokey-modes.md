# Task 1305 - Auto-Key Modes

## Goal
Provide granular auto-key behavior.

## Scope
- Global auto-key on/off.
- Per-property auto-key (pos-only, ori-only, mesh-only, 2D-only).
- Auto-key while dragging gizmos.
- Visual indicator when auto-key is armed.

## Success Criteria
- Auto-key never touches unintended channels.
