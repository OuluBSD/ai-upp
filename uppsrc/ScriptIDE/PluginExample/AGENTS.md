# PluginExample

Minimal reference package showing how to extend `ScriptIDE` through the public plugin interfaces.

This package is intentionally not linked into `ScriptIDE` by default. It exists as a concrete sample for:

- plugin registration from `.icpp`
- a dockable pane
- a preferences page
- a run-state listener

Keep it aligned with `uppsrc/ScriptIDE/PluginInterfacesGUI.h`.
